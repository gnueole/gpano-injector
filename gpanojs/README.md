# Geo Pano Injector

**Version:** 1.3.0  
**Authors:** Antigravity & Éole <hi@eole.me>

A modern, minimalist tool to read and manage EXIF data from JPEG images, specifically designed to inject **Google Photo Sphere (GPano)** XMP metadata. This ensures that your 360° panoramic images are correctly recognized and displayed by Google Photos and other compatible viewers.

## Features

*   **Drag & Drop Upload:** Easily upload JPEG images.
*   **EXIF & XMP Viewer:** View detailed metadata extracted from your images.
*   **GPano Injection:** One-click injection of standard Google Photo Sphere XMP tags.
*   **Download:** Save the processed image with a `-360web` suffix.
*   **Dark Mode:** Toggle between light and dark themes.
*   **Offline Capable:** Works entirely in the browser without a backend server.

## Documentation

For more information on the XMP metadata used, refer to the official documentation:
[Google Photo Sphere XMP Metadata](https://developers.google.com/streetview/spherical-metadata)

## How to Run

Simply open `index.html` in your web browser. No installation or server required.
